## Prerequisites

[Node.js](https://nodejs.org/en/) is required to launch.

## Installation

1. Clone the repository
2. Install packages

`yarn`

3. Serve data at http://localhost:5000/results

`yarn run api`

4. View project at http://localhost:3000

`yarn run dev`
